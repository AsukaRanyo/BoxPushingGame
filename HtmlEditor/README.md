推箱子关卡编辑器

快速使用说明：

1. 在编辑器（推荐vscode）中打开此文件夹。
2. 直接用浏览器打开 [index.html](index.html)。可以右键文件 -> 在默认浏览器中打开（或使用 Live Server 扩展）。
3. 在左侧拖拽图块到网格，或点击图块后在网格上拖动绘制。设置宽度/高度后点击重置会重建空白网格。
4. 点击“导出 CSV”会生成与 demo1.csv 相同格式（包含行序号、LevelNum、X、Y、ActorType）的文件。

注意UE5 编辑器通常需要手动刷新 Content Browser 或对目标 DataTable 执行重新导入，才能把新的 CSV 应用到 DT_GameDataTable 资产上。
